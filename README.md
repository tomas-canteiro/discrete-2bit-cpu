# Discrete 2-Bit CPU Project
## Description
This project aims to build a 2-bit Central Processing Unit using only discrete components. By setting aside the use of integrated chips, the objective to master the manual management of signal paths, logic states and fundamental circuit schematics.

## Project Status
- [ ] __Step 1__: AND Gate
- [ ] __Step 1b__: NOT Gate
- [ ] __Step 1b__: OR Gate
- [ ] __Step 1c__: XOR Gate
- [ ] __Step 2__: Half-Adder
- [ ] __Step 3__: Full-Adder
- [ ] __Step 4__: SR Latch
- [ ] __Step 5__: Clock Generation
- [ ] __Step 6__: Arithmetic Logic Unit (ALU)

## Technical Specifications
* **Power Supply:** +5V DC
* **Active Components:** BJT NPN Transistors (BC547); 5mm LEDs
* **Passive Components:** Carbon Film Resistors (270Ω)
